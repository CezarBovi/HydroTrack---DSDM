# 💧 HydroTrack — Gestão de Hidratação Diária

Aplicativo multiplataforma desenvolvido em Flutter para controle do consumo diário de água, com cálculo de IMC, metas personalizadas, histórico de consumo e lembretes.

---

## Pré-requisitos

- [Flutter SDK](https://docs.flutter.dev/get-started/install) versão 3.0 ou superior
- [Android Studio](https://developer.android.com/studio) (para rodar no Android)
- [VS Code](https://code.visualstudio.com/) com a extensão Flutter instalada

---

## Como rodar o projeto

**1. Clonar o repositório**
```bash
git clone https://github.com/seu-usuario/hydrotrack.git
cd hydrotrack
```

**2. Instalar as dependências**
```bash
flutter pub get
```

**3. Verificar o ambiente**
```bash
flutter doctor
```

**4. Rodar no Chrome**
```bash
flutter run -d chrome
```

**5. Rodar no Android (celular físico)**

Ative o Modo Desenvolvedor no celular: Configurações → Sobre o telefone → toque 7 vezes em Número da versão. Depois ative a Depuração USB em Configurações → Opções do desenvolvedor. Conecte o celular via USB, aceite o popup de depuração e rode:
```bash
flutter run
```

**6. Rodar no Windows**
```bash
flutter run -d windows
```
